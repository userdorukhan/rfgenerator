# Transformer Design & Research

This directory contains the documentation and design justifications for the custom RF step-up transformer being tested in the lab 

## Project Summary
Due trade offs between turns ratio, output voltage, and frequency range, no commercial parts met the full specification[cite: 4].
This document outlines the transition to a custom-wound solution using **Material 43 NiZn ferrite toroids** to achieve high-voltage operation (1.5 kV) across a wide 100 Hz – 10 MHz frequency range.

### Key Specifications
* **Target Output:** 1.0 - 1.5 kV with a center-tapped secondary[cite: 2].
* **Core Selection:** FT-240-43 chosen to minimize leakage inductance and prevent low-frequency saturation[cite: 14, 15].
* **Current Status:** Moving to full test suite integration with the PA107 for frequency response and phase balance verification[cite: 23].
